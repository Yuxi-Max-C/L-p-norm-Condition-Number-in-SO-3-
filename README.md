# L-p-norm-Condition-Number-in-SO-3-
We study the distribution of $L^p$-norm condition numbers of a tetrahedral measurement frame under rotations in $\mathrm{SO}(3)$. While the $\ell^2$-norm condition number remains invariant, the $\ell^1$ and $\ell^\infty$ norms vary with orientation, revealing geometry-dependent robustness beyond conventional $\ell^2$-based metrics.

# SO(3) Condition Number Dataset for Polarimetric Measurement Design

This repository provides the full dataset used in the study of condition number distributions under rotations of a tetrahedral measurement frame on the Poincaré sphere.

## 📊 Dataset

The file `tetra_SO3_L1_Linf.csv` contains:

- Rotation X (deg)
- Rotation Y (deg)
- Rotation Z (deg)
- L1 Norm Condition Number
- L2 Norm Condition Number
- L∞ Norm Condition Number

Each row corresponds to a rotated configuration of a fixed tetrahedron under SO(3).

## 🧠 Description

The dataset is generated by:

- Fixing a regular tetrahedron on the Poincaré sphere
- Applying rotations in SO(3):
  - First rotating about the Y-axis
  - Then sweeping rotations about X and Z axes
- Computing condition numbers under different norms:
  - ℓ¹
  - ℓ²
  - ℓ∞

## 🔬 Purpose

This dataset enables:

- Statistical analysis of condition number distributions over SO(3)
- Investigation of geometry-dependent robustness beyond ℓ² metrics
- Reproducibility of figures in the associated manuscript
